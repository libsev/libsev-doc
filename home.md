# About libsev
Libsev is an event loop library providing support for
* Single threaded event loops
* Thread pooled event loops
* Event callbacks
* Win32 message loops
* Asynchronous IO streams (IO completion ports)
* Timers
* Fibers
* Human input devices (game controllers)
* ...
